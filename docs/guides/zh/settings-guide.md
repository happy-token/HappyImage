# HappyImage 设置指南

设置页面分为 5 个分区，逐一说明。

## 1. 运行环境

检查系统依赖是否就绪（Bun、Git、Chrome、内置技能）。

- **绿色圆点** = 已安装且可用
- **红色圆点** = 必需但未安装
- **黄色圆点** = 可选依赖

## 2. AI 模型配置

分两个标签页：

### 执行模型

配置 HappyImage 用于计划生成、文案撰写和多轮修改的模型。当前支持 Anthropic/Claude 兼容接口。

必填字段：
- **密钥**：Anthropic 密钥（通常以 `sk-ant-...` 开头）
- **Base URL**：默认 `https://api.anthropic.com`，使用代理或兼容网关时修改
- **Model**：默认 `claude-sonnet-4-6`

### 生图模型

配置 baoyu-imagine 使用的图片生成后端。支持 OpenAI、Google Gemini、OpenRouter、DashScope、Azure OpenAI、Replicate、Z.AI、MiniMax、Jimeng、Seedream。

每个模型提供商卡片显示：
- 当前配置状态（默认 / 已保存 / 未配置）
- Base URL 和模型名称
- 密钥状态（已配置 / 未配置）

操作：
- **编辑配置**：打开弹窗填写密钥、Base URL、模型名称等
- **设为默认**：将该模型提供商设为默认图片后端

弹窗中：
- **保存配置**：只保存配置，不改变默认模型提供商
- **保存并设为默认**：保存配置并同时设为默认后端

## 3. 偏好设置

### 外观

- **深浅主题**：深色 / 浅色 / 跟随系统
- **主题颜色**：靛青 / 翡翠绿 / 玫红 / 琥珀黄 / 青蓝 / 紫罗兰
- **界面语言**：中文 / English / 日本語

### 默认配置

- **输出目录**：生成图片和项目文件的保存路径
- **默认比例**：1:1 / 16:9 / 9:16 / 4:3 / 3:4
- **跳过计划确认**：开启后自然语言输入直接开始生成，不展示计划确认步骤

### Skill 偏好

选择 Skill → 编辑默认值 → 点击"保存 Skill 偏好"。不同 Skill 有不同的可配置项（风格、布局、配色、水印等）。

## 4. 平台发布

### 平台登录态

为各平台打开浏览器登录页，扫码登录后保存会话。后续发稿复用已保存的登录态。

- 微信扫码登录 / 微博扫码登录 / X 扫码登录 / 小红书扫码登录

### 公众号账号向导

配置微信公众号的 API 发布参数（App ID、App Secret、默认排版样式、评论控制）。

### 高级浏览器配置

可折叠区域，配置 Chrome Profile 路径和各平台独立的 Chrome 启动路径。默认自动检测，一般无需修改。

## 5. 备份与恢复

- **导出配置 JSON**：将所有密钥和设置导出为 JSON 文件
- **导入配置 JSON**：从备份文件恢复全部配置
