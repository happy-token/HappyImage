# HappyImage 用户指南

欢迎使用 **HappyImage**！AI 视觉卡片生成与多平台一键发布工具。

## 文档导航

指南已按使用场景拆分为三个独立文档：

| 文档 | 适合 | 内容 |
|------|------|------|
| [快速开始](guides/zh/quick-start.md) | 新用户 | 安装、配置密钥、生成第一张图 |
| [设置指南](guides/zh/settings-guide.md) | 需要配置的用户 | 5 个设置分区逐一说明 |
| [发布指南](guides/zh/publish-guide.md) | 内容创作者 | 各平台规则、登录态管理、常见排查 |

英文文档也保留在 `docs/guides/en/`，适合英文界面或英文读者使用。

## 核心功能

- **智能卡片生成**：输入创意，选择视觉风格和配色，一键生成高清推广卡片
- **多尺寸适配**：1:1、3:4、16:9 等主流社交媒体比例
- **AI 文案生成**：按平台格式（小红书/微博/X/微信公众号）自动生成文案
- **平台预览**：1:1 还原各平台发布界面
- **会话聊天**：多轮对话式创作，所有记录自动保存
- **一键发布**：自动打包素材并唤起 Chrome 完成填写

## 平台规则速查

| 平台 | 最多图片 | 标题限制 | 正文限制 | 话题标签 |
| :--- | :---: | :--- | :--- | :--- |
| 小红书 | 18 | 20 字 | 1000 字 | 最多 10 个 |
| 微博 | 18 | 内联【】 | 2000 字 | `#双#号#` |
| X | 4 | 无 | 280 字符 | `#tag` |
| 微信公众号 | 不限 | 64 字 | 20000 字 | 不支持 |
